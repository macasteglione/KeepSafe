# KeepSafe - Protección Familiar

KeepSafe es una aplicación móvil diseñada para proteger a tu familia mientras navega por internet.
Actúa como un guardián digital que filtra contenido peligroso y ayuda a mantener un entorno seguro
en línea para niños y adolescentes.

## ¿Para qué sirve?

### Protección contra Contenido Peligroso

- Bloquea automáticamente sitios web con contenido inapropiado
- Filtra pornografía, violencia y otros materiales dañinos
- Utiliza servidores DNS seguros de CleanBrowsing Family Filter

### Control Parental

- Establece una contraseña para controlar la aplicación
- Requiere autenticación para desactivar la protección
- Funciona de manera invisible en segundo plano

### Seguridad Continua

- Se activa automáticamente al encender el dispositivo
- Persiste a través de reinicios y actualizaciones
- Protege todos los navegadores y aplicaciones

## ¿Cómo funciona?

1. **Instalación**: Descarga e instala KeepSafe desde la tienda de aplicaciones
2. **Primera Configuración**: Establece una contraseña segura
3. **Activación**: Concede permisos de administrador del dispositivo
4. **Protección Automática**: La app se activa sola y protege inmediatamente
